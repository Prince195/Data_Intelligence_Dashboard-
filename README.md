# 📊 Data Intelligence Dashboard (Excel Project)

## 📌 Project Title

**PR. Final Project – Data Intelligence Dashboard**

## 📂 Dataset

This project uses the **"Final Project Dataset"** sheet containing **200+ records of customer transactions**.

The dataset includes:

* Customer Names
* Product Details
* Transaction Dates
* Sales Amount
* Region Information

## 🎯 Objective

The main objective of this project is to build a **complete data analysis and reporting solution using Microsoft Excel**.

This includes:

* Analyzing customer transactions
* Extracting meaningful insights
* Creating interactive dashboards
* Presenting business intelligence visually

---

## ⚙️ Tools & Technologies

* Microsoft Excel
* Pivot Tables & Charts
* Data Analysis Toolpak
* Excel Formulas & Functions

## 🔑 Key Features Implemented

### 1. 📅 Date & Time Functions

Used functions like:

* `TODAY()`
* `NOW()`
* `DATEDIF()`
* `EOMONTH()`

Purpose:

* Calculate durations
* Track transaction timelines
* Monthly analysis

---

### 2. 🔍 FILTER Function

* Extracted specific records based on conditions
* Enabled dynamic multi-value filtering

---

### 3. 🎨 Conditional Formatting

* Used icons and arrows
* Highlighted trends (increase/decrease)
* Created KPI indicators

---

### 4. ⏱ Timestamp Creation

* Used `NOW()` function
* Applied iterative calculation for dynamic timestamps

---

### 5. 📈 What-If Analysis

* Scenario Manager
* Goal Seek

Used to:

* Predict sales targets
* Perform sensitivity analysis

---

### 6. 📉 Linear Regression

* Used **Data Analysis Toolpak**
* Identified relationship between variables
* Predicted trends

---

### 7. 📊 Data Analysis & Storytelling

* Identified patterns in customer behavior
* Found revenue trends
* Highlighted business insights

---

### 8. 👥 High-Value Customers (GroupBy Concept)

* Used formulas like:

  * `SUMIFS`
  * `COUNTIFS`

Purpose:

* Identify top customers
* Segment customers based on value

---

### 9. 📊 Dashboard Creation

#### Used:

* Pivot Tables
* Pivot Charts (Bar, Line, Pie)

* Total Revenue
